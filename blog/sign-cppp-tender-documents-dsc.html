How to Sign CPPP Tender Documents with DSC Token

By PDFSign  ·  Oct 2025  ·  5 min read
Home › Blog › How to Sign CPPP Tender Documents with DSC Token

The Central Public Procurement Portal (CPPP) at eprocure.gov.in is used by hundreds of central government departments for publishing and receiving tenders. Bidders must submit digitally signed documents — without a valid DSC-signed PDF, your bid is incomplete and will be rejected.

What CPPP Requires

  • A valid Class 3 DSC issued by a licensed Certifying Authority: eMudhra, Capricorn, NSDL, (n)Code, Pentasign, vsign, or Sify
  • PDF documents signed with your DSC before upload
  • The signature must be cryptographically embedded — not a scanned image

Common Mistakes That Get Bids Rejected

  • Uploading a scanned signature image instead of a digital signature
  • Using an expired DSC certificate
  • Using an incorrect certificate — always verify the certificate name matches your CPPP-registered identity
  • Modifying the document after signing — this breaks the cryptographic seal
Important: Fill in all required fields in the tender document before signing. Any change after signing invalidates the signature.

Step-by-Step: Sign CPPP Tender Documents

1Install Your Token Driver

Mandatory first step. For HYP2003 tokens, install the HyperPKI middleware. For ProxyKey (Watchdata) tokens, install the ProxyKey middleware. Restart if prompted.

2Plug In Your DSC Token

Insert your USB token and wait for Windows to detect it.

3Open PDFSign Software

Launch PDFSign from your desktop or Start menu.

4Select Your Certificate

PDFSign detects your token automatically and shows your certificate in the dropdown. If multiple certificates are present, click Refresh and select your Class 3 certificate. Confirm the name matches your CPPP-registered identity.

5Select Signing Mode and Load Document

For a single tender document, choose Single PDF mode and click Browse to load the file. If you have multiple tender documents to sign, use Batch mode and select all files together. PDFSign processes each one with the same certificate.

6Place Your Signature

Drag the signature box on the PDF preview to the designated signature area, or enter the page number and coordinates manually.

7Choose Signing Scope

Select Single Page or All Pages as the CPPP document requires.

8Click Sign and Enter Your PIN

Click Sign. Enter your token PIN. PDFSign embeds your digital signature into the document.

9Upload to CPPP

The signed file is saved as filename_signed.pdf. Set a custom output location before signing if needed. Upload the signed document to CPPP before the tender deadline.

Frequently Asked Questions

What class of DSC is required for CPPP tenders?

CPPP tenders require a Class 3 DSC. Your certificate must be issued by a licensed CA — eMudhra, Capricorn, NSDL, (n)Code, Pentasign, vsign, or Sify. Note: Class 2 DSCs are no longer issued in India, so any currently valid DSC is already Class 3.

Can I sign multiple tender documents at once?

Yes. PDFSign supports Batch mode — select all your tender documents and sign them in one operation with the same certificate.

What happens if I modify the document after signing?

Any modification after signing breaks the cryptographic seal. The signature will show as invalid when opened. Always complete the document fully before signing.
